# PoolRadar

Track your favorite pool players, venues, and tournament directors across DigitalPool tournaments.

**[https://slyfox3.github.io/PoolRadar/](https://slyfox3.github.io/PoolRadar/)**

Save favorites, see their recent and upcoming tournaments, and follow match results live. Automatically handles DigitalPool's duplicate accounts. Links to [TournamentPerf](https://slyfox3.github.io/TournamentPerf/) for performance analysis.
